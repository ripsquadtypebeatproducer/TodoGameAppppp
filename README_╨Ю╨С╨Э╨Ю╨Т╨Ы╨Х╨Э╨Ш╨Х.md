# TodoGame — обновление: новый дизайн + 4 фичи

## Что добавлено

### Оформление
- Минималистичный стиль: скруглённые карточки с мягкими границами, круглые чекбоксы
  с цветом приоритета, «воздух» между элементами, пилюли для серий/тегов.
- Светлая и тёмная темы сохранены (переключаются в Настройках).

### Новые функции
1. **Питомец-маскот** (экран «Питомец») — лисёнок/котёнок/панда/дракончик,
   рисуется программно с анимациями (счастливый прыгает, грустный оседает).
   Настроение зависит от серии дней и выполненных задач.
2. **Статистика продуктивности** (экран «Статистика») — график выполненных задач
   за неделю, метрики (всего, за неделю, серия, лучший день).
3. **Магазин наград** (экран «Магазин») — за монеты покупаются питомцы, темы,
   бейджи и аватары. Монеты и опыт (XP) начисляются за выполнение задач.
4. **Друзья** (экран «Друзья») — поиск по имени, заявки в друзья, просмотр
   статистики друга, совместные задачи (долгое нажатие на задачу → поделиться).

Все 4 экрана доступны с экрана **Профиль** (раздел «Разделы»).

## Установка базы данных
1. Открой `database/TodoGameDB_setup.sql` в SQL Server Management Studio.
2. Выполни скрипт (F5) под учётной записью с правами sysadmin.
   Скрипт создаёт БД, логин `todogame`, все таблицы и наполнение.
   Скрипт идемпотентный — можно запускать повторно.
3. Если БД уже была — скрипт сам добавит недостающие колонки (миграции).

Приложение также само создаёт/мигрирует таблицы при первом запуске
(метод `DatabaseHelper.initDatabase`), так что скрипт — для ручной настройки.

## Вход
- Администратор: `admin@todogame.com` / `admin123`
- Обычные пользователи регистрируются через экран регистрации.

## Технические заметки
- Новые поля в `Users`: Xp, Coins, UserLevel, PetType, PetName, PetHappiness, Theme.
- Новые таблицы: Friendships, TaskShares, ShopItems, UserItems.
- Кастомные View: `views/PetView.kt` (питомец), `views/BarChartView.kt` (график) —
  рисуются на Canvas, без внешних библиотек.
