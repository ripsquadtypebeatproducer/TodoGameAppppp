# TodoGame — мобильное приложение для управления задачами с геймификацией

## Данные для входа

**Администратор:**
- Email: `admin@todogame.com`
- Пароль: `admin123`

**Демо-аккаунт (с заполненными данными для скриншотов):**
- Email: `demo@todogame.com`
- Пароль: `demo123`

Демо-аккаунт автоматически создаётся при первом запуске и содержит:
- 8 активных задач + 20 выполненных
- 4 категории (Работа, Учёба, Дом, Здоровье)
- 4 привычки с заполненными тепловыми картами
- 25 записей настроения для графика
- Серия активных дней: 12
- Полученные достижения

## Функционал

### Для пользователя (USER)
- Регистрация и вход (пароли хешируются SHA-256)
- Создание/редактирование/выполнение/удаление задач
- **Создание папок прямо на экране добавления задачи** (кнопка «+»)
- Категории (папки) с цветами
- **Корзина** (доступна через Настройки) — восстановление удалённых задач
- Привычки с тепловой картой за 91 день
- Ежедневные вызовы
- Трекер настроения с графиком
- Достижения (15 шт.) и аватары (12 шт.)
- Тёмная/светлая тема

### Для администратора (ADMIN)
Панель администратора доступна через **Настройки → Панель администратора** (только для роли admin):
- **Дашборд статистики** системы: пользователи, задачи, выполнено, привычки, настроения, выполнено сегодня
- **Прогресс-бар** общего процента выполнения задач
- **Таблица лидеров** с медалями 🥇🥈🥉
- **Управление пользователями**: повышение до администратора / понижение до пользователя, удаление пользователей со всеми данными

## Настройка перед запуском

1. **SDK путь** в `local.properties` — замени `YourName` на свой путь к Android SDK (или удали файл, Android Studio создаст сам).

2. **Подключение к БД** в `app/src/main/java/com/todogame/app/database/DatabaseHelper.kt` (строки 11-13):
   - `SERVER = "10.0.2.2"` — для эмулятора (это localhost компьютера). Для реального телефона укажи локальный IP компьютера с SQL Server.
   - Логин/пароль БД: `todogame` / `TodoGame123!`

3. **База данных** — запусти `setup_database.sql` в SSMS (или таблицы создадутся автоматически при первом запуске).

## Технологии
Android (Kotlin), MVVM, Material Design, SQL Server 2019 Express, JTDS JDBC 1.3.1, Kotlin Coroutines, AndroidX LiveData/ViewModel.
